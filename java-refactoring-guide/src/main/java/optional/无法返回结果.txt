无法返回值时的三种方案:
1. throw exception     抛出checked exception会在客户端造成额外的异常处理(样版代码)
2. return null;        可能被客户端忽略, 造成NullException异常
3. return Optional<T>; 类似于抛出unchecked exception, 可能造成NullException异常
